Everything in one place

Six command tabs, live metrics, and a wizard that handles the entire install — all running locally, nothing phoning home.

⚡
Live System Stats

CPU, RAM, temperature, and disk usage updated in real time via psutil. Sparkline graphs for historical context at a glance.

STS TAB
🖥️
Terminal Runner

Execute shell commands from the browser. Full command history, Ctrl+R history search, and per-OS quick-command libraries built in.

CMD TAB
🐳
Docker Manager

List, start, stop, restart, and inspect containers without touching a terminal. Real-time container status with log tailing.

DKR TAB
📁
File Browser

Browse, read, edit, and download files from your home directory. Respects .dashignore rules for privacy.

FLS TAB
🔌
Ports & Services

See every open port and listening process. Watch custom systemd / launchd / Windows services and get alerted when they drop.

PRT TAB
📜
Script Runner

Save and run named scripts from scripts.scr. Organize into groups, run with one click, view output inline.

SCR TAB
🔒
Secure Auth

PAM authentication on Linux, scrypt-hashed passwords on macOS and Windows. Session timeout configurable from setup wizard.

AUTH
🧙
Setup Wizard

Browser-based installer handles pip deps, config, service registration (systemd/launchd/Task Scheduler), and desktop shortcuts.

SETUP.PY
🗑️
Clean Uninstall

Reads config.json to surgically remove exactly what was installed — service, shortcuts, unit files — without touching anything else.

UNINSTALL.PY

Pick your platform

Standalone builds require no Python install. Source distribution works anywhere Python 3.8+ is available.

🪟
Windows
.exe — standalone binary

Single-file executable bundled with PyInstaller. Runs on Windows 10/11 (x64). Registers as a Scheduled Task for autostart. No install wizard — just double-click.

↓  devboard-windows.exe
🐧
Linux
.deb — Debian / Ubuntu package

Installs via dpkg -i. Drops a binary to /usr/local/bin and a systemd user service. Compatible with Debian 11+, Ubuntu 22.04+, Raspberry Pi OS.

↓  devboard-linux.deb
🍎
macOS
.dmg — drag-and-drop app

Standard disk image with a .app bundle inside. Drag to Applications, right-click → Open on first launch (Gatekeeper unsigned). Supports macOS 12 Monterey and later.

↓  devboard-mac.dmg

// getting started

Up in three steps

Installers handle everything. For power users who prefer source, a full manual setup path is documented below.

⚡ Quick Install (executables)

  • Download your platform's build Grab the .exe, .deb, or .dmg from the Download section above or the GitHub Releases page.

  • Run the installer / binary Windows: double-click the .exe — a setup wizard opens in your browser.
    Linux: sudo dpkg -i devboard-linux.deb
    macOS: open the .dmg, drag the app to Applications.

  • Complete the wizard Set your port, password, and whether to register as a system service. The wizard handles everything automatically.

  • Open the dashboard Navigate to http://localhost:5000 (or your chosen port). Bookmark it — that's your dashboard from now on.

🔧 Source Install (Python)

  • Clone the repository

  • Run the setup wizard Installs flask, psutil, pywebview and registers a system service.

  • Start the dashboard After setup, the service auto-starts on boot. Or run manually any time.

  • To uninstall Reads config.json and removes exactly what was installed — service, shortcuts, and config.

# clone $ git clone https://github.com/DA-COMPUTER/devboard $ cd devboard # wizard (deps + service + shortcuts) $ python3 setup.py # manual start (optional) $ python3 dashboard.py # uninstall $ python3 uninstall.py
